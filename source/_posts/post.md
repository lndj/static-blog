# Hello World!

This is Static Blog Powered by PHP.

## Usage

1. Install

```shell
composer create-project lndj/static-blog blog --dist
```

2. Init your static-blog project

```shell
php stb init
```
3. Write a new post

```shell 
php stb new
```
