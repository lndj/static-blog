# 静态博客

有句话说的话，PHPer最后都会放弃Wordpress。

我们需要新的选择。

GitHub Pages等同类型服务的出现，为我们方便的写博客，同时具有免费、无限流量、方便部署等特性提供了好的选择。

然而，限制就是不支持解析PHP等动态脚本语言，不支持数据库！

所以Jekyll、Hexo等应运而生。

本项目灵感来源于Hexo。

# 项目说明
一个使用PHP生成静态网页的博客~
本地使用Markdown书写，快速生成Html静态文件部署。
同时支持主题、评论、搜索、标签、分类、页面、RSS订阅、站点地图等普通博客具有的功能。
> 注：评论、搜索使用第三方服务。
